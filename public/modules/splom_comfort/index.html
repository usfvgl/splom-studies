Pictured above is a scatterplot matrix visualization. It shows pairwise relationships between the variables v, w, x, y, and z as a matrix of scatterplots. You can determine the pair of variables being visualized by the row and column of the scatterplot in the matrix. The legend for class colors can be found in the lower-right corner.

For example, the first column encodes the variable z using horizontal position. The first row encodes the variable v using vertical position. Therefore, you can see the same scatterplot as before for variables z and v in the upper-left corner of this matrix.

Can you find the scatterplot for variables x and w in the matrix? If so, which variable is encoded using horizontal position and which variable is encoded using vertical position? If you can answer these questions, you understand how to interpret a scatterplot matrix.
