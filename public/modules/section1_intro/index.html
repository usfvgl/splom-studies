Section 1 Introduction

The next section will ask you to perform the following task:

Drag the black rectangle to a region for variables v and z with the most circles.

It is possible the variables v and z will be replaced with other variable names.


Training

This task requires you to find the appropriate cell in the scatterplot matrix for those pairs of variables, and then select the region that has the most circles.

Animation

Both of the examples above are static (non-animated, non-moving) scatterplots. We can also animate the scatterplot by looping continuously through the data and redrawing the circles. This allows overlapping circles in the background to eventually be redrawn in the foreground. It is important to note that the animation does not represent changes over time to the underlying data.

For example, below is the same regions as before with 25 circles:

There is a small amount of movement caused by redrawing the overlapping red circles. Compare that to the region with 60 circles:

Notice there is more movement because there are more overlapping circles, and we can see that there are some gold circles near the center that pop up occasionally that were not as visible in the static version.
